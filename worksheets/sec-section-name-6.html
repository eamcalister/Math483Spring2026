    Worksheet Mean Value Property and Maximum Principle for Harmonic Functions (2D)
    

    In this worksheet, you will prove the mean value property for harmonic functions in two dimensions using the polar form of the Laplacian.
    🔗

    Definition 4.

    A function \(u\) defined on an open set \(\Omega \subset \mathbb{R}^2\) is harmonic if \(\Delta u = u_{xx} + u_{yy} = 0\text{.}\)
    🔗
    🔗
    Assume \(u\) is harmonic in some open set \(\Omega\) with \(B_r(0)\subseteq \Omega\text{.}\)
    🔗

    1. Defining \(A(r)\) and Showing it is Constant.

    Define the circular average
    \begin{equation*} A(r) = \frac{1}{2\pi} \int_0^{2\pi} u(r,\theta)\, d\theta. \end{equation*}
    Observe that this equals the arclength average
    \begin{equation*} A(r) = \frac{1}{2\pi r}\int_{\partial B_r(0)} u\, ds. \end{equation*}
    🔗
    Our goal is to show that, under the given hypotheses, \(A(r)\) is constant.
    🔗

    (a)

    Observe, the polar Laplacian of \(u\) is given by
    🔗
    \begin{equation*} \Delta u = u_{rr} + \frac{1}{r} u_r + \frac{1}{r^2} u_{\theta\theta}. \end{equation*}
    Since \(u\) is harmonic, multiply by \(r^2\) to obtain:
    🔗
    \begin{equation*} r^2 u_{rr} + r u_r + u_{\theta\theta} = 0. \end{equation*}
    Use these facts (and integration) to prove that
    🔗
    \begin{equation*} rA''(r) + A'(r) = 0. \end{equation*}
    🔗

    (b)

    Solve this ODE. Show:
    🔗
    \begin{equation*} (r A'(r))' = 0, \end{equation*}
    hence
    🔗
    \begin{equation*} rA'(r) = C. \end{equation*}
    🔗

    (c)

    Argue that \(C = 0\) using boundedness of \(A(r)\) as \(r \to 0^+\text{.}\) Conclude \(A'(r)=0\) and \(A(r)\) is constant.
    🔗
    🔗
    🔗

    2.

    Show:
    🔗
    \begin{equation*} \lim_{r\to 0^+} A(r) = u(0). \end{equation*}
    Conclude the mean value property for a circle centered at the origin:
    🔗
    \begin{equation*} u(0) = \frac{1}{2\pi}\int_0^{2\pi} u(r,\theta)\, d\theta. \end{equation*}
    🔗

    3. Mean Value Property for Circles.

    Let \(x_0 \in \Omega\text{.}\) Using translation, show:
    🔗
    \begin{equation*} u(x_0) = \frac{1}{2\pi r}\int_{\partial B_r(x_0)} u\, ds \end{equation*}
    whenever \(\overline{B_r(x_0)} \subset \Omega\text{.}\)
    🔗
    🔗

    4. The Mean Value Property for Disks.

    Let \(x_0\in\Omega\text{.}\) Use the mean value property for circles to show that
    \begin{equation*} u(x_0) = \frac{1}{\pi R^2}\int\int_{B_R(x_0)} u(r,\theta) dA. \end{equation*}
    Hint: Do a double integral in polar coordinates in the \(d\theta dr\) order. (we started using \(R\) because we need to integrate with respect to \(r\text{.}\))
    🔗
